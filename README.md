# Classic Hangman Game (C)

A simple, command-line based Hangman game written in C. This project was developed as a fundamental programming exercise to practice loops, strings, and conditional logic.

## 🎮 Game Features
* **Classic Rules:** You have 5 lives to guess the secret word correctly.
* **Dynamic Feedback:** The game updates the hidden word in real-time as you guess the correct letters.
* **Input Validation:** Boolean logic is used to track correct/incorrect guesses and update remaining lives accordingly.

## 🛠️ How to Compile and Run
1. Open your terminal.
2. Compile the source code using GCC:
   ```bash
   gcc main.c -o hangman
3. Run the game:
   ```bash
   ./hangman
